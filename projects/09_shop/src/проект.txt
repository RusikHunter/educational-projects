Создайте приложение для управления списком покупок, которое включает возможность отслеживания стоимости и количества товаров, а также поддерживает обмен данными с внешним API для получения актуальных цен на товары.

Основные функции:
Пользователь может добавлять, редактировать и удалять товары из списка.
Используйте fetch для получения актуальных цен с открытого API (например, API цен на продукты).
Сохраняйте список покупок в LocalStorage, чтобы он сохранялся при перезагрузке страницы.
Реализуйте возможность подсчета общей стоимости товаров в списке и отображения этой информации пользователю.
Добавьте возможность фильтрации товаров по категории или по наличию.
Эта задача поможет вам применять различные навыки, включая работу с DOM, асинхронность, управление состоянием и взаимодействие с API.
